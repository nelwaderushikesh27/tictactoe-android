# 🎮 TicTacToe Game

A classic TicTacToe game built with Android & Kotlin.

## ✨ Features
- 1 Player vs AI (Minimax algorithm)
- 2 Player mode
- Score tracking
- Beautiful Material Design UI
- Smooth animations
- Sound effects
- Dark/Light mode

## 🛠️ Tech Stack
| Component | Technology |
|-----------|------------|
| **Language** | Kotlin |
| **UI** | Jetpack Compose |
| **AI** | Minimax Algorithm |
| **Architecture** | MVVM |

## 🎯 Game Modes

| Mode | Description |
|------|-------------|
| **vs AI** | Play against unbeatable AI |
| **2 Player** | Pass and play |
| **Online** | Play with friends |

## 🚀 Quick Start

```bash
git clone https://github.com/nelwaderushikesh27/tictactoe-android.git
cd tictactoe-android
./gradlew installDebug
```
